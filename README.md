# My dotfiles
## Installation
Requirements:  
`stow`  
i3: `i3`, `polybar`, `dunst`, `picom`  
zsh: `lf`, `oh-my-zsh`  
termite: `termite`  

`stow i3` - sets up i3, polybar, dunst, and picom  
`stow termite` - sets up termite  
`stow zsh` - oh-my-zsh + lfcd keybind from Luke Smith (press Ctrl + O)
